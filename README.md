# Supply Chain Cost Optimization Analysis Tool

Interview-ready Python analysis pipeline for logistics cost optimization using a Kaggle shipment-pricing dataset.

## What it does
- Ingests CSV shipment data and normalizes key logistics fields.
- Identifies top transportation cost drivers (route, carrier, geography, shipment size, urgency).
- Simulates three deterministic optimization scenarios:
  - Shipment consolidation
  - Carrier tier switching
  - Lead-time relaxation
- Produces visualizations, summary tables, and a one-page recommendation.

## Project structure
- `src/data_pipeline.py`: ingestion and canonical transformation
- `src/cost_driver_analysis.py`: baseline and cost-driver analysis
- `src/scenario_model.py`: rule-based scenario simulation
- `src/visualize.py`: static visualization generation
- `src/reporting.py`: executive summary generation
- `src/main.py`: orchestration entrypoint (`run_pipeline`)

## Run
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m src.main --input /path/to/kaggle_supply_chain.csv --output outputs
```

## Outputs
- `outputs/figures/*.png`
- `outputs/tables/*.csv`
- `outputs/recommendation_onepager.md`

## Testing
```bash
pytest -q
```
